# PerspectiveLab — Simple User Guide

**Full client handover (all OS):** **[CLIENT_HANDOVER.md](./CLIENT_HANDOVER.md)**

No Docker. No coding. Works on **macOS**, **Windows**, and **Linux**.

---

## Start

| OS | Starter |
|----|---------|
| Mac | `Start App.command` |
| Windows | `Start App.bat` |
| Linux | `Start App.sh` or `./start.sh` |

Browser: **http://localhost:8000** — keep the terminal open.

---

## First time

1. **Setup** → paste OpenRouter or OpenAI key  
2. **Workspace** → ask agents  

Key stays on her laptop in `backend/.env`.

---

## Flow

Setup → Workspace → Report → Compare / Study → Presentation → Export → Guide

Languages: EN / PT / FI.

Optional second case: set `CASE_ID=digital-inclusion` in `backend/.env`.
